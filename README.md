- Swagger da API : https://aw-hotel-api.herokuapp.com/swagger-ui/index.html#

- Java 8
- Maven
- Spring Boot
- Postgresql
- banco h2
- GIT

- Spring Data JPA
- Spring Validation

- Spring Devtools
